>_ MISSION_OBJECTIVE

In modern algorithmic trading, traditional setups suffer from proprietary data leakage to cloud LLMs, execution latency from network HTTP calls, and API reliability issues. Institutional setups strictly prohibit external transmission of active trade ideas.

Investment Matrix is a self-hosted, offline-first cryptocurrency market research and automated paper-trading platform. It maintains a zero-cloud footprint, processes tick-level data with sub-second lag, and utilizes local LLM inference for multi-agent trade thesis evaluation.

>_ ARCHITECTURE & STACK

Frontend & Backend: Next.js Web UI for the client interface with a FastAPI backend server.

Data Ingestion: WebSocket adapters stream live ticks directly into Redis Streams to act as an asynchronous backpressure buffer. A Python Writer Service then performs bulk inserts into a TimescaleDB hypertable for high-velocity candle storage and historical querying.

Inference & Execution: Celery workers invoke an Ollama container mapped to the host's GPU (e.g., Llama 3.1 8B). This ensures all model inference remains completely local. To guarantee deterministic execution, a structured JSON protocol with aggressive sanitization and Pydantic validation retry loops is enforced.

Core Engineering Challenges: We tackled high-frequency ingestion bottlenecks by separating streamers from writers. To balance latency, we developed a dual-track decision engine: a deterministic fast-lane formula engine executing in milliseconds for aggressive paper trading, paired with an asynchronous agent audit for deeper risk analysis.
